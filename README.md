Namma Kathey is an educational Android application developed using Kotlin and Jetpack Compose. The app is designed to preserve and promote the inspiring stories of Karnataka’s legendary heroes, freedom fighters, warriors, and historical leaders through an interactive digital learning experience.

This application is mainly built for students, history enthusiasts, young learners, teachers, and people interested in Karnataka’s culture and heritage. Users can explore district-wise heroes, read story-based chapters, take quizzes, earn badges, and learn history in both English and Kannada.

The main goal of Namma Kathey is to make history learning engaging, interactive, and accessible through modern mobile technology while preserving the rich cultural legacy of Karnataka.

Features
District-wise hero exploration
Interactive story chapters
Kannada and English language support
Quiz and rewards system
Achievement badges and progress tracking
Offline JSON-based content storage

This project was developed as part of an internship project focusing on modern Android application development using MVVM Architecture and Jetpack Compose.
